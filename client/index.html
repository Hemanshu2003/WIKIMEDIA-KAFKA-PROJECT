Kafka : Event Streaming Wikimedia

This is a analysis of recent changes on Wikimedia properties. You can interact with the charts to filter dynamically. Charts are updated with new events as they are fetch from the API ( localhost:3000/data-source ). Click on the "Clear all filters" link to remove all filters you have applied.

The API is the Mimic of Wikimedia event stream, This project shows the working and implementation of the basic workings of Apache Kafka.
1. Producing data to the Kafka topic.
2. Consuming data from the Kafka topic.
3. Customizing Kafka Produces and Consumers.
4. and many more concept.
